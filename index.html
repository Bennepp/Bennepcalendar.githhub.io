Interactive calendar with events, made by Bennep for proposal, enjoy. Send a pull request if you see anything wrong, I'll be surprised if there's not

HOW TO USE:

In the script.js file, you will see the events variable (at the top) it is formatted into years, months, and days. If you want to add an event called "BOD meeting" on the 9th November for example, you would do:


2024: { // Year 2024
    10: { // November (Month 8) (months start at month 0, I can't be bothered to change, sorry) (jan is month 0)
        9: "BOD meeting", 
        5: "Meeting with Bob",
    },
}, // Close the loop

What this does is creates events in 2024, in November, and on the days 9 and 5. i have provided examples in the script with comments to help you out if you want to test it out. Also, i havent tested this on mobile, im not sure if if works
